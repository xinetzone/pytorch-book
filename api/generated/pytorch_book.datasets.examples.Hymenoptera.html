pytorch_book.datasets.examples.Hymenoptera¶

class pytorch_book.datasets.examples.Hymenoptera(data_dir: pytorch_book.datasets.file.PathType = 'data/hymenoptera_data', download: bool = False)[源代码]¶

Hymenoptera 数据集 是 ImageNet 的一个非常小的子集。

有大约 120 个蚂蚁和蜜蜂的训练图像。每个类有 75 张验证图片。

参数

data_dir – 数据根目录

变量

  • ~Hymenoptera.dataset_sizes – 数据集大小

  • ~Hymenoptera.dataloaders – 数据加载器

  • ~Hymenoptera.class_names – 类名列表
