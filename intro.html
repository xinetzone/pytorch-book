项目简介¶

提供与 PyTorch 相关的优质资源。

环境配置¶

推荐使用 Anaconda3 管理 Python 环境。

本项目使用：

  • Python 版本：

!python --version

Python 3.10.0

  • PyTorch 版本：

import torch

torch.__version__

'1.11.0'

贡献¶

为项目贡献文档，需要

pip install .[doc]

还需要安装 PyTorch：

conda install pytorch torchvision torchaudio cpuonly -c pytorch

conda install pytorch torchvision torchaudio cudatoolkit=11.3 -c pytorch

文档生成：

invoke docs
